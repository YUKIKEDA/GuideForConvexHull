# 付録B アルゴリズム一覧表

本教材で扱った凸包アルゴリズムを、次元・計算量・特徴で比較した一覧表です。アルゴリズム選択の参考にしてください。

---

## 2次元凸包アルゴリズム

| アルゴリズム | 次元 | 計算量 | 特徴 | 章 |
|--------------|------|--------|------|-----|
| **Gift Wrapping**（Jarvis March） | 2D | $O(nh)$ | 直感的、実装が単純、最悪 $O(n^2)$ | 第3章 |
| **Graham Scan** | 2D | $O(n \log n)$ | 極角ソート + スタック、atan2 が必要 | 第4章 |
| **Andrew's Monotone Chain** | 2D | $O(n \log n)$ | 座標ソートのみ、数値的に安定、推奨 | 第5章 |
| **QuickHull** | 2D | 平均 $O(n \log n)$、最悪 $O(n^2)$ | 分割統治、3次元への拡張が容易 | 第7章 |
| **Chan's Algorithm** | 2D | $O(n \log h)$ | 出力敏感で最適、$h$ が小さいとき有利、実装は複雑 | 第8章 |
| **動的凸包**（素朴版） | 2D | 追加 $O(n \log n)$ | 毎回再計算、実装が簡単 | 第6章 |
| **動的凸包**（増分版） | 2D | 追加 $O(\log n)$ | 平衡木で管理、実装はやや複雑 | 第6章 |

- $n$：点数
- $h$：凸包の頂点数

## 3次元凸包アルゴリズム

| アルゴリズム | 次元 | 計算量 | 特徴 | 章 |
|--------------|------|--------|------|-----|
| **3D Gift Wrapping** | 3D | $O(nh)$ | 稜線ベース、2D の拡張 | 第13章 |
| **3D QuickHull** | 3D | 平均 $O(n \log n)$、最悪 $O(n^2)$ | 最遠点で分割、再帰 | 第14章 |
| **増分構築**（Incremental） | 3D | 期待 $O(n \log n)$ | 点を1つずつ追加、衝突グラフで効率化 | 第14章 |
| **Randomized Incremental** | 3D | 期待 $O(n \log n)$ | 点の順序をランダム化 | 第14章 |

---

## 高次元凸包

| アルゴリズム | 次元 | 計算量 | 特徴 | 章 |
|--------------|------|--------|------|-----|
| **QuickHull**（Qhull 等） | $d$ 次元 | 出力依存 | 汎用的、ファセット数は $O(n^{\lfloor d/2 \rfloor})$ まで | 第17章・第18章 |
| **Gift Wrapping** | $d$ 次元 | $O(n \cdot f)$ | $f$ はファセット数、最悪で指数関数的 | 第18章 |

---

## 出力形式・用途別の目安

| 用途 | 推奨アルゴリズム | 理由 |
|------|------------------|------|
| 一般的な 2D 凸包 | Andrew's Monotone Chain | 実装が簡単で数値的に安定 |
| $h$ が小さい 2D | Chan's Algorithm | $O(n \log h)$ で最速 |
| ストリーミング（点が逐次追加） | 動的凸包（$n$ が小さい場合は素朴版） | 再計算より増分更新が有利な場合も |
| 3D 凸包 | Qhull（scipy）または増分構築 | 実装コストと信頼性のバランス |
| 高次元 | Qhull | 実績のあるライブラリを利用 |
| 学習・理解 | Gift Wrapping → Graham Scan → Andrew's | 直感的な順序で理解を深める |

---

## 計算量のまとめ

| 記法 | 意味 |
|------|------|
| $O(n)$ | 入力サイズに比例 |
| $O(n \log n)$ | ソートと同程度 |
| $O(nh)$ | 点数と凸包頂点数に比例（出力敏感） |
| $O(n \log h)$ | 2次元凸包の理論的下界に一致（最適） |
| $O(n^2)$ | 二重ループ程度 |
| $O(n^{\lfloor d/2 \rfloor})$ | 高次元のファセット数の上界 |
