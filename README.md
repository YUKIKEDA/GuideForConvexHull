# 凸包計算アルゴリズム 包括的学習教材（Guide for Convex Hull）

Qhull などの凸包計算アルゴリズムを、理論から実装まで習得するためのカリキュラムです。初学者から上級者まで、段階的に学べる構成になっています。

---

## 概要

このリポジトリでは、2次元・3次元の凸包（convex hull）を求める代表的なアルゴリズムを解説し、可視化用の Python スクリプトを提供します。

- **凸包とは**：平面上（または空間内）の点集合を「輪ゴムで囲んだ」ときにできる凸多角形（凸多面体）のことです。
- **学ぶ意義**：衝突検出、パターン認識、GIS、ロボティクスなど、多くの分野で応用されています。

## 使い方

1. **順番に読む**  
   前の章の内容が次の章の前提になっているため、`Chapter01` から順に進めることを推奨します。

2. **手を動かす**  
   各アルゴリズムは、解説を読みながら自分で実装すると理解が深まります。

3. **可視化で確認する**  
   アニメーション用の `animation.py` が含まれる章では、次のように実行して動作を確認できます。

   ```bash
   cd Chapter03   # または 04, 05, 06, 07, 08, 13, 14
   python animation.py
   ```

   （実行には Python 3 および必要なライブラリ（例：matplotlib, numpy）が必要です。）

4. **退化ケースを試す**  
   共線点・重複点・点数が少ないケースなど、境界条件で挙動を確認すると実装の穴に気づきやすくなります。

5. **Qhull と比較する**  
   自作実装の結果を Qhull の出力と突き合わせて検証することを推奨します。
